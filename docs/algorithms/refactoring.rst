Refactoring
-----------

**Header:** ``mockturtle/algorithms/refactoring.hpp``

The following example shows how to refactor an MIG using the Akers synthesis
method.

.. code-block:: c++

   /* derive some MIG */
   mig_network mig = ...;

   /* node resynthesis */
   akers_resynthesis resyn;
   refactoring( mig, resyn );
   mig = cleanup_dangling( mig );
